## Módulo de Filtros (Daniel Elias Di Marco Borges)

Este módulo se encarga de mostrar una serie de preguntas al usuario para poder mostrarle los lugares que más se acorden a las respuestas de las preguntas por parte del usuario.

ejemplos de preguntas:


¿cual es su presupuesto destinado a todo el plan?
¿tiene alguna discapacidad?
¿quiere salir del país?
¿en donde está ubicado?
¿exterior o interior?


|¿Qué hace?|
------------
Lee los lugares creados.

Recibe las respuestas del usuario.

Aplica filtros (presupuesto, tipo de plan,etc.).
