## Descripción del Proyecto
PlanMatch es una plataforma que ayuda a los usuarios a encontrar el plan o viaje ideal utilizando su presupuesto económico como filtro principal. El sistema analiza gustos, economía y preferencias para recomendar actividades locales o viajes más grandes.

---

## Características Principales
- Sistema de autenticación
- Preguntas dinámicas para filtrar opciones
- Resultados inteligentes según presupuesto
- Detalles completos de cada lugar sugerido
- Constructor de itinerarios personalizados
